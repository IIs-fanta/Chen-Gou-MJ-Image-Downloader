# 陈狗MJ图片下载器

这是一个专为 Discord 网页版中的 Midjourney 图片设计的浏览器插件，提供便捷的图片下载和链接复制功能，让您更高效地管理和分享 Midjourney 生成的精彩作品。

## 功能特点

- 🖼️ **自动识别**：智能识别 Discord 页面中的 Midjourney 图片
- 💾 **快速下载**：为每张图片添加"保存原图"按钮，一键下载高清图片
- 🔗 **便捷复制**：提供"复制链接"功能，轻松获取图片 CDN 地址
- 🎨 **直观界面**：按钮上显示小图标，视觉效果更加友好
- 🔄 **实时监控**：自动检测页面变化，为新加载的图片动态添加功能按钮
- ✅ **操作反馈**：提供清晰的保存和复制成功提示
- 🚀 **轻量化设计**：代码精简高效，不影响浏览器性能

## 安装说明

### Chrome 浏览器安装方法

1. 打开 Chrome 浏览器，输入 `chrome://extensions/` 并按回车
2. 在右上角开启"开发者模式"（切换开关）
3. 点击左上角"加载已解压的扩展程序"按钮
4. 选择本插件的文件夹（包含 manifest.json 的文件夹）
5. 插件成功安装后，会显示在扩展程序列表中，并有"陈狗MJ图片下载器"的标识

### Firefox 浏览器安装方法

1. 打开 Firefox 浏览器，输入 `about:debugging#/runtime/this-firefox` 并按回车
2. 点击"临时加载附加组件..."按钮
3. 选择本插件文件夹中的 manifest.json 文件
4. 插件成功安装后，会显示在临时附加组件列表中

> **提示**：Firefox 临时附加组件在浏览器重启后会失效，如需长期使用，需要进行正式发布流程。

## 使用方法

1. 安装完成后，打开 Discord 网页版并登录您的账号
2. 浏览包含 Midjourney 图片的聊天频道或私信
3. 每张 Midjourney 图片下方会自动显示两个操作按钮：
   - 绿色按钮（带下载图标）：点击可直接下载图片到本地
   - 紫色按钮（带链接图标）：点击可复制图片的 CDN 链接到剪贴板
4. 操作成功后，会有短暂的提示信息显示

## 注意事项

- 本插件仅在 Discord 网页版（https://discord.com/）中生效
- 插件需要网络权限以获取和处理图片资源
- 部分浏览器可能需要额外授权才能访问剪贴板功能
- Chrome 浏览器的开发者模式安装和 Firefox 的临时附加组件在浏览器重启后可能需要重新加载
- 请确保您有权利下载和使用 Midjourney 生成的图片，遵守相关服务条款

## 技术说明

本插件基于浏览器扩展开发标准构建，主要文件和功能说明：

- `manifest.json`：插件配置文件，定义插件信息、权限和资源
- `content.js`：核心功能实现脚本，包含图片识别、按钮添加和功能逻辑
- `icons/logo.ico`：插件图标资源

核心技术点：
- 使用 MutationObserver 监听 DOM 变化，实现动态内容检测
- 正则表达式匹配 Midjourney 图片链接格式
- 浏览器原生 API 实现文件下载和剪贴板操作
- 自定义 CSS 样式确保按钮与 Discord 界面和谐融合
